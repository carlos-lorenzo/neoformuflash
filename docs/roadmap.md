# Four-Milestone Roadmap

## Milestone 1: Foundation
- Supabase schema, RLS, and RPC for forking.
- Auth and public profile pages.
- Basic browse and read-only public views.

## Milestone 2: Authoring and Study
- CodeMirror editor with Markdown and LaTeX.
- Notes, decks, and exam creation flows.
- FSRS edge integration with review queues.

## Milestone 3: AI and RAG
- Chunking and embedding pipeline in Supabase Edge.
- pgvector search, context assembly, streaming inference.
- Auto-generation of decks and exams.

## Milestone 4: Discovery and Optimization
- Public discovery feeds and SEO polish.
- AdSense-ready layout slots.
- Stripe billing integration, entitlements, and ad suppression.
- Performance tuning, cache strategy validation, and analytics.
